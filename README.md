# company-directory-rust
## About
My second project made independently while **attempting** to learn **Rust**. Written after completion of **Chapter 8.3** of **The Rust Programming Language** and everything **before Quiz 2** in **Rustlings**. Project written for studying and proof of knowledge.

## Running the program
1. **Clone the repository**:
```text
git clone https://github.com/njhotch04/company-directory-rust.git
```
2. **Navigate to the folder**:
```text
cd company-directory-rust
```
3. **Build and Run**:
```text
cargo run
```
## Usage Demo
When you run the program, it should run similar to this:

```text
--- Welcome to the Company Directory! ---
Please enter a command. (ex: help)
add

Enter department: Repair

Please enter a command. (ex: help)

quit
```
